# SentryX Email Signatures

Hosted email signatures for the SentryX team.

## How to install

1. Open your link below
2. 2. Click the "Copy signature" button
   3. 3. Paste into Gmail → Settings → Signature (or your email client's signature section)
      4. 4. Save
